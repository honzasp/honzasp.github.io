Personal projects

For further information about my projects, including links to repositories and papers, please see my web.

dancerank.cz (web) (2016)
dancerank.cz is a web that aggregates results from dancesport competitions in the Czech Republic. It is written in asynchronous Python and relies on MongoDB for storage and Redis for caching.
dort (renderer) (2016)
dort is a physically-based renderer heavily influenced by pbrt described in Pharr and Humphreys: Physically-based rendering. The major features are advanced geometry representations (dort can easily handle tens of millions of triangles in little memory), multiple shading models (BSDFs) and various rendering algorithms (direct lighting, path tracing, IGI and SPPM). As a special feature, dort can import Minecraft worlds into scenes, utilizing a special geometric structure that allows efficient ray-tracing of cubic voxels. The rendering engine is written in modern C++ and embeds a Lua interpreter so that scenes are scripted in Lua.
Spiral (programming language) (2015)
Spiral is a simple functional language inspired by Scheme. It has first-class functions, tail-calls, module system, both mutable and immutable data structures and a mark-and-sweep garbage collector. Compilation is separated into multiple stages using two intermediate languages. I implemented the compiler and wrote a paper about it as my maturita (high-school leaving exam) task. The compiler is written in Rust and the runtime in C++.
libext2 (filesystem driver) (2016)
libext2 is a Rust library for manipulating ext2 filesystems. It supports all common filesystem operations (both reading and writing) and integrates a FUSE interface. This was my first semestral program at the university.
Krt (browser game) (2014)
A game running in browser written in CoffeeScript, using modern Web APIs. Up to four players control circular vehicles and shoot each other in a procedurally generated system of caves.
Totem Destroyer (game) (2009)
A game inspired by a well-known Flash game, written in C++ using SDL with OpenGL for 2D graphics and Box2D for realistic physics. The player has to carefully destroy all blocks of a "totem" and prevent precious idols from hitting the ground.
